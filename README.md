## Quick Customization

### Change the accent color?
1. Open `styles.css`
2. Find `--color-accent: #fbbf24;`
3. Replace with your color

### Add a project?
1. Duplicate a project card
2. Update title, description, tech stack
3. Add image and link

### Update about section?
1. Replace the 3 paragraphs with your story
2. Keep the same structure
